# WhereAreMyRepos

*Version 0.1.0*

---

My fun little tool written in Ruby to see where you have any local Git repositories tucked away.

Sorry if the tool isn't the most "elaborete" or "fancy" my experiance with
Ruby is single file programs not structured projects, hope you enjoy anyway!

The tool is very fast, it uses the systems native shell to execute a shell command. Now I know your probably like "Just write a shell script!"
 I don't know Bash or Powershell, this is a thin wrapper basically.
